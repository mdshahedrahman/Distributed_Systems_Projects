How to Run the program: 
-----------------------
1. Navigate to the directory 'Assignment 2 - Vector Clock Algorithm' containing the files 
server.py
machine_1.py
machine_2.py
machine_3.py

2. Open 4 terminals.

3. In one of the terminals, run the server using the following command:
python3 server.py

4. In the other 3 terminals, activate the machines using only one of the following commands in each of the terminals respectively:
python3 machine_1.py
python3 machine_2.py
python3 machine_3.py

5. You can terminate the processes at any time using CTRL+C
