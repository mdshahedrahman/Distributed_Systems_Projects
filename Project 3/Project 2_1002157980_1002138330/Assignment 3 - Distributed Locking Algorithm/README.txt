How to Run the program: 
-----------------------
1. This is a 3-node distributed system, implementing a distributed locking mechanism on a shared memory. Navigate to the directory 'Assignment 3 - Distributed Locking Algorithm'.

2. Initially generate files in the shared_directory as:
$ bash gen_file.sh

3. Run <node_1> script as:
$ python3 n1.py

4. Run <node_2> script as:
$ python3 n2.py

5. Run <node_3> script as:
$ python3 n3.py

6. Finally, run the <client> script as:
$ python client.py
